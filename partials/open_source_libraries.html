Open Source Libraries

Go open-source packages

June 2013-Present

Senior Principal Software Engineer & Architect

I have been writting and releasing open-source libraries for the Go programming language since shortly after I started programming in Go in June 2013.

Currently, I have released hundreds of them. But some highlights include:

  • Concurrency and Parallelism
    • a Go implementation of Erlang's and the OTP's supervision tree
  • Data Formats
    • a Go implementation of the JSON data format
      • (This is meant to be an alternative to Go's built-in "encoding/json" package.)
    • a Go implementation of the Unicode UTF-8 data format
      • (This is meant to be an alternative to Go's built-in "utf8" package.)
    • a Go implementation of the twtxt data format
  • Monads
    • a Go implementation of optional types — also known as option types and maybe types
    • a Go implementation of nullable optional types — also known as option types and maybe types
    • a Go implementation of expiring optional types — also known as option types and maybe types — which is useful for dealing with caching and cache expiration
  • Natural Language Processing (NLP)
    • A Go implementation of the Porter Stemming algorithm, including the 'departures'
  • Network Protocols
    • a Go implementation of the finger protocol
    • a Go implementation of the gemini protocol
    • a Go implementation of the mercury protocol
    • a Go implementation of the telnet protocol
    • a Go implementation of the (secure) telnets protocol
  • Social Webn
    • a Go implementation of the Mastodon API
  • Testing
    • a Go package for producing arbitrary data of different types to help with testing
    • A Go package for creating "middleware" for Go's built-in "database/sql" package
