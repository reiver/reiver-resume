Sol-Social (contract)

July 2024 - August 2024

Solidity Developer

Sol-Social is an open-source library for secure smart-contract development of blockchain-based social-media networking.

  • I created, from scratch, a contract that provides the ability to add a primary-key to a contract.
  • I created, from scratch, a contract that provides the ability to add an advanced control mechanism to a contract, where a set of accounts (the delegates) can be granted access to specific functions.
  • I created, from scratch, a contract used to represent a person, organization, service, application, etc, on a social-media network.

The work was done using the Solidity programming language.
