  • I created the backend and frontend, from scratch, for a charitble donations plaform. I also included an implementation of OpenSocial.
  • I worked on the backend and frontend for an e-commerce application for provincial park in British Columnia.
  • I worked on the backend and frontend for a poker game.
  • I worked on the backend and frontend for an e-commerce application for a store that sells kids' clothing, toys, baby gear, and more from parents.

The work was done using the PHP and JavaScript programming languages, and the Postgres and MySQL database servers.
