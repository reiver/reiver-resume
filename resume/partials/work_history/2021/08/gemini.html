Gemini (contract)

August 2021 - November 2021

Senior Go Developer

The Gemini-protocol is an application-layer Internet communication protocol for accessing remote documents, similar to HTTP and Gopher.

(This work was done for the company that became #SpaceHost.)

  • I implemented, from scratch, the server-side of the Gemini-protocol, for Gemini backends.
  • I implemented, from scratch, the client-side of the Gemini-protocol, for Gemini frontends.
  • I implemented, from scratch, the a Gemini server that supports the separation of encryption system from the server system.

This work was done using the Go (i.e., Golang) programming language.
