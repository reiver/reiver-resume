Blockchain Game (contract)

May 2018 - August 2018

Solidity Developer

This was a blockchain-based video game built on the Ethereum blockchain-network. (The exact video game isn't mentioned due to an NDA.)

  • I created, from scratch, contracts that managed the information about the video game characters.
  • I created, from scratch, contracts that managed the information about the NPCs.
  • I created, from scratch, contracts that managed battles between video game characters.
  • I created, from scratch, contracts that managed world progression.
  • I created, from scratch, a backend that intefaces with the contracts and made it accessible to the frontend.

The work was done using the Solidity and Go (i.e., Golang) programming languages.
